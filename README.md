

# ❤️ Heart Stroke Prediction using Machine Learning

A Machine Learning web application built with **Streamlit** that predicts the likelihood of heart disease based on a user's medical information. The model analyzes various health parameters and provides an instant prediction to help demonstrate the use of ML in healthcare.

## 🚀 Features
- Predicts heart disease risk in real time
- Interactive and user-friendly Streamlit interface
- Trained using a Machine Learning model
- Fast and accurate predictions
- Easy to deploy and use

## 🛠️ Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## 📊 Input Parameters
- Age
- Gender
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- Oldpeak
- ST Slope

## ▶️ Run Locally

```bash
git clone https://github.com/yourusername/heart-stroke-prediction.git
cd heart-stroke-prediction
pip install -r requirements.txt
streamlit run app.py
```

## 🎯 Project Goal
This project demonstrates how Machine Learning can be integrated with Streamlit to create an interactive application for predicting heart disease risk. It is intended for educational purposes and to showcase ML deployment skills.
